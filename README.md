🌸 Flower Classification using Deep Learning Overview

This project uses a CNN (Convolutional Neural Network) to classify different types of flowers from images using deep learning.

Dataset

The model is trained on a flower image dataset containing categories like:

Daisy 🌼
Dandelion 🌻
Roses 🌹
Sunflowers 🌻
Tulips 🌷

Images are organized in folders by class and used for training and validation.

Features
Image classification of flowers
Deep Learning using TensorFlow/Keras
Supports custom image input
Tech Stack

Python, TensorFlow, Keras, NumPy, Matplotlib

How to Run
git clone https://github.com/1Maazansari/flower-classification.git
cd flower-classification
pip install -r requirements.txt
jupyter notebook
Output
Predicts flower category from input image
